Exibir Desconto para Pagamento a-vista Woocommerce

Código para exibir uma porcentagem de desconto (%) na página de produto do Woocomerce, utilizando o tema OceanWP.

O valor de desconto padrão é 10% (10). Para alterar a porcentagem de desconto, basta aterar o valor de "$desc" (ex.: $desc = 15 / 100; para 15% de desconto).

Para editar o estilo basta utilizar a class CSS ".price_disc_cash". Ex.: .price_disc_cash { color: #d50000; font-size: 18px; }
